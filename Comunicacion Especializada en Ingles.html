Layout

They are a set of containers where we can place many elements according to the design of our application, for example within a Layout you can place buttons, images, forms, texts, etc. All these must have an order and harmony among themselves within the design, for this we must have chosen a Layout first and then place the elements we need within it.

Interface

The user interface of your app is everything that the user can see and interact with. Android offers a variety of previously compiled UI components, such as structured design objects and UI controls that allow you to compile the graphical user interface for your app.

Java

Java is a programming language and a computing platform that was first commercialized in 1995 by Sun Microsystems.

Methodology

The methodology refers to the set of rational procedures used to achieve the objective or range of objectives that governs a scientific investigation, a doctrinal exposition or tasks that require specific skills, knowledge or care.

GridView

The GridView is an extremely flexible control for displaying multicolumn tables.

RecyclerView

This widget is a container for displaying large data sets that can be scrolled very efficiently while maintaining a limited number of views.

ListView

shows a list of items (each item in the list can be made up of various visual controls) and allows us to interact with its elements (select, delete, reorder, etc.)

Fragments

represents a behavior or part of the user interface in a FragmentActivity. You can combine multiple shards into a single activity to create a multi-panel UI and reuse a shard in different activities.
